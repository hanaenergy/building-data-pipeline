# Automated Building Data Enrichment Pipeline

An automated ETL pipeline that transforms messy building data into a trusted analysis-ready dataset by enriching it with public climate and building reference data.

## Business Problem

Many building owners and consulting firms still manage building information in Excel spreadsheets.

Different address formats, missing values, and duplicated records make climate-based energy analysis slow and error-prone.

This project aims to automate the workflow—from data cleaning to climate data integration—to improve data quality and reduce manual work.
